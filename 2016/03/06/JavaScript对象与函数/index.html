JavaScript对象与函数

题记

这两天简单看了下JavaScript，做下笔记，备忘。

JavaScript对象

与Java等其他语言不同，JavaScript对象其实一个动态的属性包，即属性集合，每个属性由属性名称与属性值组成。

对象的创建
  1. 文字形式
    var obj = {key: value};
  2. 构造形式
    var obj = new Object();
    obj.key = value;

对象的属性值可以是任意JavaScript对象，属性名称则只能是字符串。

对象属性值的访问
  1. obj.key (. 操作符，通常被称为’属性访问’)
  2. obj[“key”] ([]操作符，通常被称为’键访问’)
六种基本类型

JavaScript中有六中基本类型(并不是对象):

  1. string
  2. number
  3. boolean
  4. null
  5. undefined
  6. object
内置对象(对象子类型)
  1. String
  2. Number
  3. Boolean
  4. Object
  5. Function
  6. Array
  7. Date
  8. RegExp (正则表达式)
  9. Error

    其实是JavaScript中的一些内置函数，可以直接使用函数的构造方式创建对象。使用原型中提供的方法。

原型（prototype）

JavaScript对象有一个指向原型的链，当试图访问该对象的属性时，如果在该对象的属性集合中未找到，则搜索该对象的原型，如果还未搜索到，则会搜索该对象原型的原型，直到找到或者，原型的指向为null。

例子（自定义函数，并创建对象）： 

function MyObj(){   
  this.name = "lili";   
}   
MyObj.prototype.sayHello = function(){console.log("hello, I'm " + this.name)};   
var myObj = new MyObj();
myObj.sayHello();
myObj.name = "lucy";
myObj.sayHello();
JavaScript函数

JavaScript中，函数是一等公民，所谓一等公民，即可以声明直接调用。JavaScript中，函数也可以认为是对象。
可以对函数进行以下操作：

  1. 赋值给变量。
  2. 指定为一个对象的属性值。
  3. 函数作为参数传递。
  4. 函数作为函数返回值返回。
  5. 使用字面值来创建函数。

在HTML中创建顶层变量时，会将变量创建为window实例的属性。

作为回调函数

setTimeout(function(){console.log("Hey, guy!")},3000);

函数中的this

JavaScript中this指向的对象（或称为执行上下文）。与函数的调用位置，调用方式有关。

一段误解的例子: 

  function foo(num){  
    console.log("foo: " + num);  
    //记录foo被调用的次数  
    this.count++;  
  }  
  foo.count = 0;  
  var i;  
  for(i = 0; i<5; i++){  
    foo(i);  
  }  
  console.log(foo.count);//0

是的，输出是0.
再来一段： 

function foo(num){  
  console.log("foo: " + num);    
  //记录foo被调用的次数  
  this.count++;  
}  
count = 0;  
var i;  
for(i = 0; i<5; i++){  
  foo(i);  
}  
console.log(count);//5
//-----------------通过obj 调用
var obj = {count: 1};  
foo.call(obj,1);
console.log(obj.count);//2

上面两个例子应该能大致解释清楚this的含义。即指向当前运行的上下文，也就是调用者。
当直接执行函数的时候，this则指向全局环境（默认绑定）。当在严格模式(strict mode)及”use strict”时，则不能将全局对象应用到默认绑定，因此 this会绑定到undefined。

立即执行表达式

先上段代码 A：

var foo = (function(){
  var secret = "secret";
  return{
    get_secret: function(){
      return secret;
    },
    new_secret: function(new_secret){
      secret = new_secret;
    }
  };
}());
console.log(foo.get_secret());

JavaScript中的立即执行表达式：

(function(){/todo/})(); //这个可以用
(function(){/todo/}()); //推荐使用这个
由于函数被包含在()内，所以被解析成为表达式，而在表达式后面加上()时，将会执行该表达式。就像函数名后面加上()会执行该函数一样。
所以代码 A中的例子，foo其实一开始就被赋值为立即执行表达式中返回的对象： 

return{
  get_secret: function(){
    return secret;
  },
  new_secret: function(new_secret){
    secret = new_secret;
  }
};

所以拥有 get_secret, new_secret 这两个函数属性。可以调用。
这么做也是闭包的很好的应用。
